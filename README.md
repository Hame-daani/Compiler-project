# Calculator
A Compiler to evaluate mathmatic expressions.  
Both **integer** and **float** numbers and also **negative** numbers are supported.  
for more information see **Project_Description.pdf**.

## Supported Operation
* Addition `+`
* Subtraction `-`
* Multiplication `*`
* Division `/`
* Exponentiation `p` -- **Note the Character!**
* Parantheses `( )`

## Dependecys
* Ply

## Examples
```
Input Exp >>>> 2+2
Result Is -> {4}
```
```
Input Exp >>>> -2 * 3
Result Is -> {-6}
```
```
Input Exp >>>> (2p2)p2
Result Is -> {16} 
```
